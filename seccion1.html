Tecnologia:

La naturaleza abierta de la web ofrece increíbles oportunidades para los desarrolladores que desean crear sitios web o aplicaciones online.
Para sacar el máximo provecho de las capacidades de la web, necesitas saber cómo usarlas.
Explora los contenidos siguientes para aprender más sobre varias tecnologías web.

  • Basicas

  • HMTL

    HyperText Markup Language (HTML) es el lenguaje utilizado para describir y definir el contenido de una página web en un formato bien estructurado.

  • CSS

    Cascading Style Sheets (CSS) u Hojas de estilo en cascada se utilizan para describir el aspecto o presentación del contenido en una página web.

    Hypertext Transfer Protocol (HTTP) o Protocolo de Transferencia de Hipertexto (HTTP) es un protocolo de la capa de aplicación para la transmisión de documentos hipermedia, como HTML.

Definicion de una Web:

World Wide Web, o simplemente Web, es el universo de información accesible a través de Internet, una fuente inagotable del conocimiento humano.

El componente más usado en el Internet es definitivamente el Web. Su característica sobresaliente es el texto remarcado, un método para referencias cruzadas instantáneas. En la mayoría de los Sitios Web, ciertas palabras aparecen en texto de otro color diferente al resto del documento. Por lo general, este texto es subrayado. Al seleccionar una palabra o frase, uno es transferido al sitio o página relacionado a esa frase. En algunas ocasiones hay botones, imágenes, o porciones de imágenes que pueden activarse mediante un clic. Si Usted mueve el apuntador sobre el contenido del documento y el apuntador cambia a un símbolo con una mano, eso indica que Usted puede realizar un clic para ser transferido a otro sitio.

Usando el Web, se tiene acceso a millones de páginas de información. La exploración en el Web se realiza por medio de un software especial denominado Browser o Explorador. La apariencia de un Sitio Web puede variar ligeramente dependiendo del explorador que use. Así mismo, las versiones más recientes disponen de una funcionalidad mucho mayor tal como animación, realidad virtual, sonido y música..

Distribucion en sitios WEB

Cuando hablamos de distribución de contenidos dentro del diseño web, nos referimos a la forma de organizar los elementos que componen nuestro sitio web, puede tratarse de el texto empleado en la página web, el cual debe tener cierta estructura y coherencia; o de cualquier otro elemento que contenga información relevante para el usuario.

Para mejorar la usabilidad de nuestro sitio, debemos definir un esquema de organización, funcionamiento y navegación del sitio, y esto no se refiere a algo relacionado con el aspecto gráfico del sitio o la estética del mismo, si no del concepto, su arquitectura. Debemos tomar en cuenta la topoligía del sitio , los elementos de multimedia usados y varios aspectos más.
